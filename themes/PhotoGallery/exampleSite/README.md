# PhotoGallery Example Site

This is the example site for the PhotoGallery Hugo theme.

## Quick Start

```bash
# Clone the theme
git clone https://github.com/YOUR_USERNAME/hugo-PhotoGallery.git

# Enter the example site
cd hugo-PhotoGallery/exampleSite

# Run hugo server
hugo server --themesDir ../..
```

## Structure

- `content/` - Example content
- `static/` - Static assets
- `hugo.toml` - Site configuration
