# Screenshots Directory

Add theme screenshots here:

- `screenshot-home.png` - Homepage screenshot (recommended: 1500x1000 or 3:2 ratio)
- `screenshot-detail.png` - Post detail page screenshot
- `tn.png` - Thumbnail image (900x600 for Hugo themes site)

## Screenshot Requirements

For submission to [Hugo Themes](https://themes.gohugo.io/):
- Homepage screenshot should be named `screenshot.png`
- Thumbnail should be named `tn.png`
- Both should be placed in this directory
- Recommended size: 1500x1000 for screenshot, 900x600 for thumbnail
